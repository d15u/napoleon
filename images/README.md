# Images Folder

Place your image assets here:

## Required Images:

### Favicon (Browser Tab Icon)
- **Files**: 
  - `favicon-16x16.png` - 16x16px
  - `favicon-32x32.png` - 32x32px
  - `apple-touch-icon.png` - 180x180px (for iOS)
  - `android-chrome-192x192.png` - 192x192px (for Android)
  - `android-chrome-512x512.png` - 512x512px (for Android)
- **Format**: PNG
- **Usage**: Browser tab icon, bookmarks, home screen shortcuts

### Logo
- **File**: `logo.png`
- **Recommended size**: 200x60px (or similar aspect ratio)
- **Format**: PNG with transparent background preferred
- **Usage**: Navbar logo

### Banner Images (Responsive)
- **Mobile**: `banner-mobile.jpg` - 375x120px
- **Tablet**: `banner-tablet.jpg` - 768x150px  
- **Desktop**: `banner-desktop.jpg` - 1200x180px
- **Format**: JPG or PNG
- **Usage**: Full-width banner background (automatically selects based on screen size)

## Optional Images:

### App Screenshot
- **File**: `app-screenshot.png`
- **Recommended size**: 375x812px (iPhone screen size)
- **Format**: PNG
- **Usage**: Phone mockup display

### Testimonial Avatars
- **Files**: `avatar1.jpg`, `avatar2.jpg`, `avatar3.jpg`
- **Recommended size**: 80x80px
- **Format**: JPG or PNG
- **Usage**: Customer testimonial photos

## Notes:
- All images will have placeholder fallbacks if not provided
- Images are optimized for web display
- Use descriptive filenames for better organization
